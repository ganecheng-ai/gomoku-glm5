# 五子棋游戏开发计划

## 项目概述
使用 Python 和 Pygame 开发一个精美的五子棋游戏，支持简体中文界面，操作与经典五子棋游戏一致。

## 技术栈
- **语言**: Python 3.8+
- **GUI框架**: Pygame
- **构建工具**: PyInstaller
- **CI/CD**: GitHub Actions
